# Brain games application
## Run App.java to play brain games
#### You will see the menu with the game list  
##### ---------------------
Please enter the game number and press Enter.  
1 - Greet  
2 - Even  
3 - Calc  
4 - GCD  
5 - Progression  
6 - Prime  
0 - Exit  
##### ---------------------
#### pick the one you like and enjoy
